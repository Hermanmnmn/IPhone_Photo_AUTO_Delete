# 📱 iPhone 照片「閱後即焚」自動清理系統

這是一個專為 iPhone 使用者設計的**照片自動清理系統**。開啟後，所有新拍的照片、影片或螢幕截圖，都會在 **7 天後** 自動移至相簿的「最近刪除」檔案夾，讓您的手機相簿永遠保持乾淨，不再被無用的暫存照片佔滿空間！

本系統完全使用 iOS 內建的**「捷徑 (Shortcuts)」**與**「自動化」**功能，**不需下載任何第三方 App**，所有資料安全地儲存在您自己的 iCloud 雲碟中，隱私 100% 安全。

---

## ✨ 核心特色
* 🔴 **一鍵背景開關**：可在 iOS 18 控制中心直接切換開關，並彈出優美的系統通知。
* ⚡ **全自動運作**：設定後，每次關閉相機自動標記，每週定時背景清理，無感體驗。
* 🔒 **隱私安全**：完全無第三方伺服器，資料儲存在您個人的 iCloud 雲碟。
* 🛠️ **自動初始化**：首次匯入直接執行即可，系統會自動建立所需的檔案夾，無須手動設定。

---

## 🚀 快速安裝與設定 (只需 3 步)

### 步驟 1：下載並匯入捷徑
請在您的 Mac 桌面上找到 **`iPhone自動刪除捷徑`** 資料夾，並透過 **AirDrop** 將以下 3 個捷徑傳送到您的 iPhone 上點擊安裝：
1. 🔄 **`1_切換自動刪除模式`**：用於開啟或關閉自動清理。
2. 📸 **`2_自動標記新照片`**：在後台自動為新照片排程 7 天後刪除。
3. 🗑️ **`3_執行自動刪除清理`**：自動將已到期的照片移至「最近刪除」相簿。

---

### 步驟 2：設定 iOS 個人自動化
為了讓系統在背景自動運作，我們需要設定兩個 iOS 內建的自動化觸發器：

#### 2.1 拍照後自動標記 (相機關閉時執行)
1. 打開 iPhone 的 **「捷徑」App**，切換至底部的 **「自動化」** 分頁。
2. 點擊右上角的 **「+」**，選擇 **「App」** 作為觸發條件。
3. 點擊「選擇 App」，勾選 **「相機」**，然後點擊「完成」。
4. 勾選 **「已關閉」**，並**取消勾選**「已開啟」。
5. 選擇 **「立即執行」** (Run Immediately)，並確認 **關閉**「執行時通知」。
6. 點擊「下一步」，搜尋並選擇 **「執行捷徑」**，將捷徑設定為 **`2_自動標記新照片`**，儲存即可。

#### 2.2 每週定時自動清理 (每週執行一次)
1. 在捷徑 App 的「自動化」分頁，點擊右上角的 **「+」**，選擇 **「特定時間」**。
2. 設定您希望自動清理的時間（例如：**每週日 02:00**）。
3. 選擇 **「立即執行」**，並確認 **關閉**「執行時通知」。
4. 點擊「下一步」，搜尋並選擇 **「執行捷徑」**，將捷徑設定為 **`3_執行自動刪除清理`**，儲存即可。

---

### 步驟 3：開啟「刪除不詢問」隱私權限
iOS 預設在刪除相片時會彈窗詢問。為了實現全自動的「閱後即焚」無感清理，請進行以下設定：

1. 打開 **「捷徑」App**，在 **「捷徑」** 分頁中找到 **`3_執行自動刪除清理`**。
2. 點擊卡片右上角的 **「...」** (三個點) 進入編輯畫面。
3. 點擊螢幕最下方的 **「資訊 (i)」** 圖示 (iOS 18 中位於螢幕正底部的圈圈 i)。
4. 選擇 **「隱私」** 分頁。
5. 找到 **「照片」**，將權限改為 **「允許存取」**，並確保將 **「刪除不詢問」** 開關 **開啟 (啟用)**。

---

## 🎛️ 進階推薦：加入控制中心 (iOS 18)
您可以像控制手電筒一樣，隨時在控制中心切換「自動刪除模式」：

1. 在 iPhone 螢幕右上角向下滑動打開 **「控制中心」**。
2. 長按空白處進入編輯狀態，點擊底部的 **「新增控制項目」**。
3. 搜尋並選擇 **「捷徑」**，點擊它並將捷徑指定為 **`1_切換自動刪除模式`**。
4. 設定完成！現在點擊此按鈕，就會在背景直接切換狀態，並在螢幕頂部通知您：
   * `🟢 自動刪除模式已開啟 (7天自動刪除)`
   * `🔴 自動刪除模式已關閉`

## ❓ 常見問題 FAQ

#### Q1. 為什麼清理照片時，手機依然跳出「允許捷徑刪除這張照片嗎？」
請參考**步驟 3**，進入 `3_執行自動刪除清理` 的隱私設定中，確認已將照片權限設定為「允許存取」，且「刪除不詢問」開關已啟用。

#### Q2. 拍照關閉相機後，沒有收到「新照片已標記」的通知？
* 系統只會在自動刪除模式為 **「開啟」** 狀態下才會標記照片。請先確認已點擊開啟。
* 檢查 iPhone 的「設定」 > 「通知」 > 「捷徑」，確認已允許捷徑發送通知。

#### Q3. 如果在 7 天到期前，我自己把照片手動刪除了，系統會報錯嗎？
不會。系統設計了防錯機制，如果發現清單中的照片已被您手動刪除或不存在，清理捷徑會安全地忽略並自動從排程清單中移除該記錄，不會有任何彈窗或報錯。

#### Q4. 如何徹底移除這個系統？
1. 點選 `1_切換自動刪除模式` 關閉系統（收到「🔴 自動刪除模式已關閉」通知）。
2. 在捷徑 App 中刪除這兩個自動化觸發器。
3. 打開 iPhone 內建的 **「檔案」App**，刪除 `iCloud 雲碟/Shortcuts/` 底下的 **`AutoDeletePhotos`** 資料夾即可。
